# Baremaps - Network Information Center (NIC)

This module is a collection of tools to READ NIC files and have some stream of NIC Objects in return.

## Data sources

- [RIPE](https://ftp.ripe.net/ripe/dbase/ripe.db.gz)
- [AFRINIC](https://ftp.afrinic.net/pub/dbase/afrinic.db.gz)
- [APNIC - inetnum](https://ftp.apnic.net/pub/apnic/whois/apnic.db.inetnum.gz)
- [APNIC - inet6num](https://ftp.apnic.net/pub/apnic/whois/apnic.db.inet6num.gz)
- [ARIN](ftp://ftp.arin.net/pub/rr/arin.db.gz)
- [ARNIN - noauth](ftp://ftp.arin.net/pub/rr/arin-nonauth.db.gz)
- [LACNIC](ftp://ftp.lacnic.net/pub/stats/lacnic/delegated-lacnic-extended-latest)
